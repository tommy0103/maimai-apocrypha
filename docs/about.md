# 关于 Maimai Apocrypha

Maimai Apocrypha 是一个**非官方**的 maimai 剧情汉化归档站点，致力于为中文玩家提供完整的剧情翻译与双语对照。

## 数据来源

所有原始剧情数据均来自 [maimai.sega.jp](https://maimai.sega.jp/)，通过自动化脚本抓取并整理。

## 翻译说明

- **自动化翻译**：使用 ai 进行辅助翻译
- **社区校对**：欢迎通过 PR 或表单提交更准确的翻译
- **双语对照**：所有页面均保留日语原文，可折叠查看

## 贡献方式

如果你发现翻译有误或想补充翻译，可以通过以下方式参与：

- **GitHub PR**：直接修改 `raw_data/*.zh.json` 文件并提交 PR
- **表单提交**：填写[贡献翻译表单](/contribute)（无需 GitHub 账号）
- 后续会提供直接在网页修改的接口，需要登录 github 帐号

## 免责声明

- 本项目为**非官方**项目，与 SEGA 无任何关联
- 所有内容仅供学习交流使用
- 如涉及版权问题，请联系我们删除相关内容

## 联系方式

如有问题或建议，欢迎在 [GitHub Issues](https://github.com/tommy0103/maimai-apocrypha/issues) 中提出。
