# Sky Street 区域3（スカイストリートちほー3）
::: info 区域简介
开始发生各类事件的 Sky Street。这是一个由试图交错过去与未来的两人之意图，以及另一人所怀抱的秘密，所引发的故事。
:::

<summary>查看日语原文 (Original Text)</summary>

</details>
