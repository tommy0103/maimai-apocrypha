# BLACK ROSEちほー9（blackrose9 区域）
::: info 区域简介
身为女警察官的Ritz。由于不被周围认可，她心中产生了空隙，而某个人甜美的〈话语〉乘虚而入。在Ellie的陪伴下，Owl无法追查案件时，Nick能否找到被委托寻找的失踪者？这是一个关于雾之城中出现的游行所引导的故事。
:::
::: tip 一些小信息
追逐恶魔游行的见习侦探及其伙伴的区域
:::

<summary>查看日语原文 (Original Text)</summary>

> 女性警察官であるリッツ。周りに認められないと言う彼女の心の隙に入り込んできたのは、ある人物の甘美な〈言葉〉だった。エリーの付き添いでアウルが事件を追えない中、ニックは依頼された失踪者を見つける事が出来るのか？ これは、霧の街に現れるパレードがいざなう物語。
