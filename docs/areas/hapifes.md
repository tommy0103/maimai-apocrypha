# 欢乐祭典区域（ハピフェスちほー）
::: info 区域简介
那是在大吉岭的收获祭结束之后。Ras买完东西回来，在街上听到一个惊人的消息，兴奋地开始说道：「喂喂！Chiffon、Salt！你们知道能实现愿望的星星吗！？」「哎呀，是童话故事吗？呵呵，这样的Ras也很可爱呢。」「就像个小孩子一样呢。」Ras鼓起脸颊说：「真是的！我在说正经话啦！」但两人道歉后，她便开始讲述每十年举办一次、庆祝星星诞生的节日——『星之生诞祭』。
:::

> それはダージリンの収穫祭が終わった頃の事。買い出しから帰って来たラズが、街ですごい事を聞いたの！と、興奮した様子で話し始めました。「ねね！ シフォンちゃん、ソルトちゃん！二人は願いを叶えてくれる星って知ってる！？」「あら、おとぎ話ですか？うふふ、そんなラズさんもかわいいですわ」「ちっちゃい子みたいです」ラズは「もう！真面目な話だよ」とほっぺたをぷくっと膨らませましたが、2人が謝ったので、10年に一度開催される星の誕生を祝うフェスティバル、『星の生誕祭』の事を話し始めました。
</details>

星之生诞祭。那是庆祝被人们寄托愿望的『愿之星』每十年重生一次的节日。镶嵌马赛克瓷砖的美丽广场被发光的矿石照亮，宛如移动游乐园。人们在堆满彩色糖果和食物的摊位、表演有趣节目的大道艺人和演奏家之间开心地穿梭。Ras、Chiffon和Salt也一边用空中漂浮的许多星星创造喜欢的星座一边走着，突然欢呼声响起。烟花绽放，从聚集在会场的人们手中的灯笼里，矿石一齐升上天空。Ras也取下手中灯笼的金属扣，里面的矿石染成红色，轻轻飘上天空。中央巨大的『愿之星』也从花环中释放，为了新的十年升向天际。
</div>
        
<details class="raw-text" style="margin-top: 15px; border-top: 1px dashed var(--vp-c-divider); padding-top: 10px;">
<summary style="cursor: pointer; opacity: 0.6; font-size: 0.85em;">查看日语原文</summary>
<div style="font-size: 0.9em; margin-top: 8px; color: var(--vp-c-text-2);">
星の生誕祭。それは人々の願いを託された『願い星』が十年に一度生まれ変わる日を祝うフェスティバル。モザイクタイルの美しい広場は光る鉱石に照らされ、さながら移動遊園地の様相だ。人々は屋台に山と積まれたカラフルなお菓子や食べ物、楽しいショーを見せる大道芸人や演奏家の間を楽しそうに行き来している。ラズ、シフォン、ソルトも空に浮かぶたくさんの星で好きな星座を作りながら歩いていると、わっと歓声が上がった。花火がうち上がり、会場中に集まった人々のランタンから鉱石が一斉に空に登っていく。ラズも、手にしていたランタンの金具を外すと、中の鉱石が赤く染まり、ふわりと空に舞い上がった。中央の巨大な『願い星』もガーランドから放たれ、新たな十年の為に天へ登って行った。
