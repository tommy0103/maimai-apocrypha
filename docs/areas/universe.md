# 宇宙すてーしょん（universe 区域）
::: info 区域简介
作为新店铺宣传的一部分，Shama和Milk将在宇宙站区域举办直播！她们乘坐宇宙船Shama牛奶号飞向宇宙，口中说着「好期待直播呢！」「真的呢！」。面对袭来的种种困难、混乱和麻烦，她们能否凭借爱、魔法和牛奶的力量克服呢？敬请期待！
:::

<summary>查看日语原文 (Original Text)</summary>

</details>
