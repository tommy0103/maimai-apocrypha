# 派对区域（パーリィちほー）
::: info 区域简介
Ras正身处困境。他呼喊着『Chiffon酱，Salt酱，呜呜呜～～～！』，眼睁睁看着星球诞辰祭的会场渐行渐远。驾驶鲜红色跑车的乙姫兴奋地说『好期待呢～』，但Ras哪有心思理会。就这样糊里糊涂地被带往的星球，原来是个全年都沉浸在『宴会』喧嚣中的行星。
:::
<details class='raw-text'>
<summary>查看日语原文 (Original Text)</summary>

> ラズは今まさに窮地に立たされています。「シフォンちゃん、ソルトちゃん、ふえええ～～～っ！」視界から遠ざかっていくのは、星の生誕祭の会場です。真っ赤なスポーツカーの運転席から、乙姫が「楽しみじゃの～」と話しかけてきますがそれどころではありません。ラズがよくわからないまま連行されていくその星は、一年中「宴」で賑わう惑星なのでした。
</details>

ダージリンの王族として、特別ゲストに招かれていたエリザビーは、高らかに「宴」開幕を宣言した。<br>ここはパーリィちほー。年中「宴」が開催され、宇宙中の舞手が集結する星。<br>ステージ上では曲にのって様々なステップが繰り広げられ、会場は徐々にヒートアップしていく。<br>勇んでステージに上がった乙姫とラズだったが、彼女達のダンスを見ていた観客のテンションはなぜか下がるばかり。一人二人と別のステージに移動して、とうとう誰もいなくなってしまった。<br>人気のなくなったステージから降りた乙姫はプリプリと怒っていたが、やがてため息をついた。<br>「……なぜじゃ！ わらわの舞がダメだとでも言うのか！？」「……乙姫ちゃん」<br>肩を落とす二人の耳に「わあっ！！」と言う歓声がきこえ、そちらの様子を伺うと、ステージ上で華麗なブレイクをキメたくま達の姿が見えた。双子であるかれらの動きはピッタリと息が合い、楽しそうな観客も、やがて動きにシンクロしていく。その様子を見ていた乙姫とラズは目を輝かせて「あれじゃ(だね)っ！」と頷きあうと、再びステージに上がるのだった。
