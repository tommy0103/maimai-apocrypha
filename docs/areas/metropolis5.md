# 大都会区域5（メトロポリスちほー5）
::: info 区域简介
由黑姬亲自制作并直管的机构，即实质上管理大都会的『五摄家』。五摄家各自拥有『女忍者部队』，每隔数年便会举行一次选拔有资格入队者的御前比试。究竟谁能成功通过考验，获得光荣的入队资格呢？
:::
<details class='raw-text'>
<summary>查看日语原文 (Original Text)</summary>

> 黒姫自ら製作した直属の機関、それがメトロポリスを実質仕切っている『五摂家』だ。五摂家はそれぞれに『くノ一部隊』を抱えており、数年に一度、新たに入隊資格がある者を選抜する御前試合が行われる。見事試験を突破し、栄えある入隊資格を得られるのは一体誰か？
</details>
