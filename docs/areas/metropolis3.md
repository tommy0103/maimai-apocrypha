# Metropolis区域3（メトロポリスちほー3）
::: info 区域简介
『你们能选择的道路只有一条。那就是在24小时内交出这个星球。我们是奉「主之命」的正当回归者。』突然，覆盖Metropolis全回线和天空的屏幕上出现了一名少女，她宣告道。与之相对的是继承了灭亡「人类」细胞的唯一仿生人「黑姬」。在24小时后的总攻之前，提出的投降条件就是交出「黑姬」。
选择的是投降，还是残酷的大战。

:::
<details class='raw-text'>
<summary>查看日语原文 (Original Text)</summary>

> 『貴方達の選択できる道はただ一つ。24時間以内にこの星を明け渡す事だけだ。 我々は「主の命」を受けた正当な帰還者である。』 突如メトロポリスの全回線と、空を覆うスクリーンに映った少女は告げた。 相対するのは、滅びさった「人」の細胞を受け継ぐ唯一のヒューマノイド「黒姫」。 24時間後の総攻撃の前に示された、降伏の条件は「黒姫」の引き渡しだった。
選ぶのは降伏か、過酷な大戦か。

</details>
