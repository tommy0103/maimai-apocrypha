# 高天原区域2（高天原ちほー2）
::: info 区域简介
从废弃村落的火灾中，借助水龙之力勉强逃脱的Nigi一行人，在下一个村落又听到了不祥的传闻。村民们声称是鬼怪枯竭了水源，Nigi迫于无奈，只好不情不愿地开始调查。
:::
::: tip 一些小信息
这是Nigi一行人到访的下一处村落。村民们正因为理应丰饶的神山水源枯竭而犯愁。
:::
<details class='raw-text'>
<summary>查看日语原文 (Original Text)</summary>

> 廃村での火災から、水龍の力で何とか逃れたニギ一行。しかし、次の集落でも不穏な噂が。鬼が水を枯らしているのだと村人たちに訴えられ、ニギは渋々捜査をする事に。

::: tip 一些小信息
ニギ一行が訪れた次の集落。村人は、豊かなはずの御山の水源が枯れ、困っていました。
:::
</details>

因深信不疑的Mikumari没有拯救自己而产生的怨恨与悲伤，将神山水源地彻底冻结，并向四周弥漫着不祥的气息。<br>不久，一位长发倒竖的『鬼神』挡住了Nigi一行人的去路。<br>其邪气异常强大，一声咆哮便令周围更加寒冷。就连Amahiko的水龙也被冻结，Shinkyō的火焰也被寒冰扑灭。<br><br>就在此时，黑暗中与鬼神对峙的『心声』，忽然流入了Nigi他们的意识中。<br>仿佛受到了谁的引导一般。<br>Nigi所见的，是其心中满溢的、仅因失去孩子而产生的悲伤。得知眼前的鬼神原本是人类后，Nigi一时无法出手，事态进一步恶化。<br>就在这时，一道断断续续的微弱声音传来。<br><br>『尊贵的大人。请驱散这充满悲伤的瘴气。如此一来，我的声音便能再次传达了吧。』<br><br>那是Mikumari的声音。她那怜悯女子、试图呼唤的声音，一直被寒冰与瘴气阻隔，直到此刻才终于传到。<br>就在这时，一直跟在Nigi身后的Menō向前一步。<br>「今夜是满月。悲伤的污秽，也将在月光中得到净化吧。Nigi大人，请务必使用我的力量。」<br>解放月之力、漂浮于空中的Menō，将手中宝玉举向天空。从宝玉中释放的力量化作光芒，被吸入Nigi的剑中。<br>当剑绽放光芒的刹那，Nigi的衣装从白色逐渐染上月之眷属的暗色。将加护之力承载于剑上，Nigi斩开了黑夜。<br>重现的月光驱散了周围的污秽，Mikumari也随之伴着铃声现出身形。<br>Mikumari渡过水面，优雅地伸出手。鬼神虽停止了行动，却如解脱束缚般缓缓走近，就在她触碰那只手的瞬间——面具卷起庞大的瘴气，轻轻脱落了。<br><br>——片刻之后。<br>回过神来，满月正照耀着青色的水面。女子身上的瘴气也大半消散，如今看来只是昏过去了而已。<br>水源地恢复了原状，Nigi一行人正在照顾昏迷的女子。<br>唯一令人在意的是，被斩裂的女子瘴气，似乎和面具一起被吸入了某处，不过当时谁也没有察觉，那正是Kurotsune所为。

信じていたミクマリが救ってくれなかった恨み、悲しみは御山の水源地を凍り付かせ、周囲に禍々しい気を満たす。<br>やがて、ニギ達の行く手をふさぐように、長い髪を逆立てた『鬼神』があらわれた。<br>その邪気は強く、一声吠えると周囲を更に凍てつかせていく。アマヒコの水龍さえ凍り付き、シンキョウの炎も氷でかき消されてしまった。<br><br>その時、暗闇の中で対峙する鬼神の『こころ』がつい、とニギ達に流れ込んで来た。<br>まるで誰かの導きのように。<br>ニギの見たその心中は、子供を失った悲しみだけに満たされていた。目の前の鬼神が元人間である事を知って手出しができなくなり、さらに事態は悪くなる一方だ。<br>そんな時、とぎれとぎれのかすかな声が聞こえてきた。<br><br>『尊き御方。この悲しみに満ちた瘴気を払ってください。さすれば我が声も再び届くでしょう』<br><br>それは、ミクマリの声だった。女を哀れに思い呼びかける声は、氷と瘴気にさえぎられ今まで届かなかったのだ。<br>その時、ニギの後ろにひかえていた小さなメノウが進み出る。<br>「今宵は満月。悲しい穢れも、月の光に浄化されるでしょう。ニギ様、どうか私の力お使いください」<br>月の力を解放し空に浮かんだメノウは、手に持つその玉を空に掲げた。玉より解放された力は光となってニギの剣に吸い込まれていく。<br>剣が光を放った刹那、ニギの衣装が白から月の眷属の闇色に染まって行く。加護をその剣に乗せ、ニギは夜の闇を切り裂いた。顔を出した月があたりの穢れを払い、鈴の音と共にミクマリが姿を現す。<br>ミクマリは水面を渡り、たおやかな手を差し出した。鬼神は動きを止めていたが、憑物が落ちたようにゆっくり歩み寄り、その手にふれた瞬間だ。面が膨大な瘴気を巻き上げてぽろりと落ちた。<br><br>―――暫しの後。<br>気付けば、満月が青い水面を照らしている。女の体を包む瘴気も大半が失われて、今は気を失っているだけのようだ。<br>水源地は、元に戻り、気を失った女性を介抱するニギ達の姿があった。<br>一つ気がかりな事は、切り裂いた女の瘴気が、面と共にどこかに吸い込まれたように見えた事だったが、それがクロツネの仕業とは、この時は誰も気づかなかった。
