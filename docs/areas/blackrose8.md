# BLACK ROSE区域8（BLACK ROSEちほー8）
::: info 区域简介
围绕着Owl侦探事务所的多样人际关系。伙伴、养父、往昔的友人……以及宿敌。与Owl相关的两位人物——『搭档Nick』与『怪盗Hugh·Rue』。这是关于他们的一段小小往事。
:::

<summary>查看日语原文 (Original Text)</summary>

> アウル探偵事務所に関わる多種多様の人間模様。仲間、養父、過去の友。……そして宿敵。アウルに関係する二人の人物。『相棒ニック』と『怪盗ヒュド・ルー』これは、彼らにまつわる少しだけ昔の物語。
</details>
