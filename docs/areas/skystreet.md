# スカイストリートちほー（skystreet 区域）
::: info 区域简介
艺术、体育、音乐等各个领域的年轻才俊汇聚于天空街道。
在那里，俗称「Yozora」的四个专精于不同才能的团队齐聚一堂，构建了独特的街区风貌。
某夜，天空大厅的大型屏幕突然激烈闪烁。抬头仰望的四个人头顶上，无数蝴蝶璀璨飞舞，当这不可思议的光景消失时，他们的左手上出现了蝶形的奇异痣痕……。
蝴蝶赋予了他们回溯时间的「时间跳跃」能力，极大地改变了各自的命运。
:::
<details class='raw-text'>
<summary>查看日语原文 (Original Text)</summary>

> 芸術、スポーツ、音楽とあらゆる分野の若い才能が集まるスカイストリート。
そこには、通称「ヨゾラ」と呼ばれる各才能に特化した４つのチームが集まり、独自の街の様相を築いている。
ある夜、スカイホールの大型ビジョンが突如激しく輝いた。見上げた4人の頭上に無数の蝶が煌々と舞い、その不可思議な光景が消えた時、彼らの左手には蝶の形の奇妙な痣が…。
蝶は時を遡る「タイムリープ」の力を彼らに与え、それぞれの運命を大きく変えていく。
</details>
