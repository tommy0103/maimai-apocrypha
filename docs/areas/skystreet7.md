# スカイストリートちほー7（skystreet7 区域）
::: info 区域简介
反复进行时间跳跃，终于抵达的未来。若能触及纷乱丝线的根源，或许这好不容易才握在手中的『现在』也将不复存在。面对这沉重的现实，所有人都陷入了沉默。
:::
::: tip 一些小信息
已经无需再畏惧『重来之前的世界』了。
:::
<details class='raw-text'>
<summary>查看日语原文 (Original Text)</summary>

> タイムリープを繰り返し、ようやくたどり着いた未来。絡まった糸の根本にたどり着ければ、ようやく掴んだ『現在』もなかった事になるかもしれない。その重い現実に、全員黙り込んだ。

::: tip 一些小信息
『やり直す前の世界』を恐れる必要はもうないんだ
:::
</details>
