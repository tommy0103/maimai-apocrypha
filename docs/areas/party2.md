# 派对区域2（パーリィちほー2）
::: info 区域简介
乙姫和Ras来到的是一家外观相当复古的游戏中心。但那里陈列着一种最新锐的格斗游戏，能将自身的参数反映到游戏角色上进行对战。在各星猛者聚集的这般情境下，乙姫的胜负心熊熊燃烧起来。她不听Ras的制止，毅然报名，但是……？
:::
::: tip 一些小信息
在宴会上获胜的乙姫和Ras，决定在解散前观光这个星球，但是…？
:::
