# 天界ちほー6（heaven6 区域）
::: info 区域简介
在诺格，主要存在着四大兽人族。继承了达纳之力的伟大四种族，击退了蛮族，各自精进力量，一直和平地治理着这片土地。然而，当圣地开始被黑色的污秽侵蚀时，神之子孙们的力量逐渐衰弱，无法再召唤出他们另一形态的“兽形神”。为了查明世界异变的原因，Vates们决定前往达纳神殿。
:::
<details class='raw-text'>
<summary>查看日语原文 (Original Text)</summary>

> ノーグには大きく分けて4つの獣人族が存在している。ダーナの力を受け継ぐ偉大な4種族は、蛮族を退け、それぞれの力を研鑽してこの地を平穏に治めてきた。しかし、聖なる地が黒い穢れに侵され始めた頃、神の子孫たちはその力を弱め、自らのもう一つの姿である「獣形神」を呼びだせなくなる。ウァテス達は世界の異変の理由を知る為、ダーナの神殿を目指す事になった。
</details>
