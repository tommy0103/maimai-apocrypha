# メトロポリスちほー6（metropolis6 区域）
::: info 区域简介
终于，帝国军与 Metropolis 军之间的全面战争拉开了序幕。Metropolis 的机体们各自怀抱着思绪，凝视着悬浮在黑暗天空中的战舰。在黑云中闪耀的它们显得神圣，而可能多达数万的白色敌军，则宛如天降使者一般。
:::
