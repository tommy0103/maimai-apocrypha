# 天界区域5（天界ちほー5）
::: info 区域简介
某一天，Iofi为了前往合唱队的教练那里，匆忙穿过走廊时，看到Mika正望着窗外，轻轻叹了口气。说起来她最近没什么精神呢……Iofi试着问Mika原因――“在这个走廊，能听到圣歌队的歌声。我觉得很美妙，每天听着听着，不知不觉就想，如果我也能那样唱歌就好了……”――Mika这样想着。Mika紧握着裙摆，害羞地回答。啊，这位只知道作为支柱的责任，甚至认为自己连这样微小的愿望都无法实现……想着想着，Iofi超越了怜悯，涌起了愤怒。“――对了，Mika大人。我们来开音乐会吧！交给我，我会召集最好的乐队成员！”“等、等一下！？”“我好期待呢～！哇～……”“走掉了……Iofi真是的。没问题吧。”
:::
<details class='raw-text'>

> ある日の事、イオフィがコーラス隊のコーチに向かう為回廊を急いでいると、ミカが外を見て、ふぅとため息をついている。そういえば最近元気がなかったわね……とイオフィがミカに理由を尋ねてみると―「この回廊、聖歌隊の歌声が聞こえてくるんです。素敵だなと思って毎日聴いてる内に、私もあんな風に歌えたら、なんて……」―つい思ってしまったのです。とミカはスカートの裾をきゅっと握りながら恥ずかしそうに答えた。ああ、この方は柱としての責務しか知らず、ご自分はこんな些細な願いすらかなわないと思ってるんだわ……。考える内、イオフィは憐れみを通り越して怒りがわいてきた。「―そうだわ、ミカ様。音楽会を開きましょう！お任せください、最高のバンドメンバーを揃えますわ！」「ちょ、ちょっと！？」「こうごきたいですわ～！わ～……」「行っちゃった……イオフィったら。大丈夫かしら。」

## 楽曲ストーリー (Stories)

在聚光灯照射的舞台上。观众的目光一齐投向自己，Mika的胸口因羞涩和紧张而痛苦。紧紧闭上眼睛，一次深呼吸。乐队的演奏充满会场，她开始歌唱。一旦声音滑出，之后音乐就会引领她。和Iofi一起创作的歌词，是Mika现在真实的心情。是的，歌曲是为了给予勇气。给Mika，也给这个世界的所有人。会场融为一体，她理解了为什么如此被音乐吸引。为有如此凝聚大家心灵的事物感到高兴，Mika继续歌唱。

スポットライトが照らした舞台の上。一斉に観客の目が自分に注がれ、ミカの胸は恥ずかしさと緊張で苦しくなる。きゅっと目をつむり、一度深呼吸。バンドの演奏が会場を満たして、彼女は歌い出した。一度声が滑り出せば、後は音が手を取ってくれる。イオフィと一緒につくった歌詞は、ミカの今の素直な気持ちだ。そう勇気をくれるために歌がある。ミカにも、この世のすべての人達にも。会場が一体になり、なぜあんなに音楽に惹かれたのか理解できた。こんなにみんなの心を束ねる物がある事に嬉しくなって、ミカは歌い続けた。
