# BLACK ROSE 区域 4（BLACK ROSEちほー4）
::: info 区域简介
一个失去记忆、被孤儿院收留的少年，与向他伸出援手的炼金术师Tristan相遇。毫无交集的两人究竟如何创立了侦探事务所？这是关于Aul与挚友的离别，以及为根除黑蔷薇病而迈出炼金术侦探第一步的两篇支线故事。
:::
<details class='raw-text'>
<summary>查看日语原文 (Original Text)</summary>

> 記憶を失い孤児院に保護された少年と、彼に手を差し伸べた錬金術師トリスタン。接点の無かった二人が探偵事務所を創設するまでにはどのような物語があったのか？アウルのかけがえのない友人との別れと、黒薔薇病を絶つ為に錬金術探偵の一歩を踏みだすまでの2編のサイドストーリー。
</details>

<div style="opacity: 0.9; line-height: 1.7;">
失去记忆的少年被教堂孤儿院保护，从此在那里长大。他结识了同样境遇的朋友Holly，本以为平静的生活会一直持续。然而，这生活因月光下恶魔化的孤儿们而突然终结。Holly为了保护少年而倒下。少年试图从更多袭来的孤儿手中救出Holly，在恐惧和混乱中无意识地炼成了喷出火焰的魔方阵。火焰迅速蔓延，少年丢下不再动弹的Holly，逃进了夜晚的森林。他拼命爬上一棵平时和Holly玩耍的树，看到了熊熊燃烧的教堂。少年茫然地望着那景象，直到夜幕褪去。
</div>
        
<details class="raw-text" style="margin-top: 15px; border-top: 1px dashed var(--vp-c-divider); padding-top: 10px;">
<summary style="cursor: pointer; opacity: 0.6; font-size: 0.85em;">查看日语原文</summary>
<div style="font-size: 0.9em; margin-top: 8px; color: var(--vp-c-text-2);">
記憶をなくした少年は教会の孤児院に保護され、以来そこで育った。ホリーと言う同じ境遇を持つ友人も出来、ずっと続くと思っていた穏やかな生活。しかしそれは月光の元悪魔化した孤児達によって突如幕を下ろす。少年をかばい倒れるホリー。更に襲い来る孤児たちからホリーを助けようとした少年は、恐怖と混乱の中無意識に炎を噴き上げる魔方陣を練成してしまう。炎は勢いよく燃え広がり、動かなくなったホリーを置いて少年は夜の森に逃げ出した。普段ホリーと遊んでいた木に辿り着き必死によじ登ると、燃え盛る教会が見えた。少年はその光景を夜が白むまで呆然と眺めていた。
</div>
