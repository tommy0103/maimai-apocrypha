# ドラゴンちほー2（dragon2 区域）
::: info 区域简介
这是一个“龙与人类”成对诞生的世界。灵魂伙伴即使出生在不同地点，也注定会相遇，并共度一生。但有一位少年，他不仅找不到伙伴，甚至所有遇到的龙都讨厌他……？这便是Ask为寻找伙伴龙而踏上旅程的故事。
:::
<details class='raw-text'>
<summary>查看日语原文 (Original Text)</summary>

> ここは「ドラゴンと人間」が対となって生まれてくる世界。魂のパートナーは、別々の場所で生まれても必ずめぐり逢い、一生運命を共にする。しかし少年は、唯ひとりパートナーに出会えないどころか、出会うドラゴンにことごとく嫌われる体質で……？これはパートナードラゴンを探して旅する、アスクの物語。
</details>

野良ドラゴンや、盗賊の類から逃げ回りつつも、山頂近くに何とかたどり着いたアスク。<br><br>一息つこうと腰を降ろすと、いつの間にか眠ってしまっていた。バリボリと言う音で目を覚ますと、そこにいたのは、オレンジ色でフワフワの羽毛の大きな鳥。何すんだ！この野郎とカバンを奪い返すが、既に食料は殆ど食い尽くされてしまっている。<br>「モットナンカクワセロ」というオレンジ鳥に、半泣きで「もうねえよ！」と言うと、しょんぼりした様子でアスクを見上げた。<br>しょんぼりしたいのはこっちですけど!? と思いつつも、諦めてアスクが立ち去ろうとすると、鳥がヒョコヒョコ後をついて来る。<br>「お前を連れて行く義理はないんだけど!? 大体俺はレジェンドドラゴンを探して……って、なんだよ、お前？」<br>翼があるので鳥かと思ったが、良く見ると巨大な鍵爪のある両腕を振り回して走ってくる。<br>「ワスレタ……でもカワリニコレヤルゾ」<br>そう言って、首から下げた赤い石をアスクに掲げる。<br>「お前、まさかそれ契約の……」<br>アスクがそう言いかけた時だ。「ギャアアア！」と空気を割く叫び声があたりに響いた。<br>話しながら歩いていたアスクは、森の様子が暗くなっている事に気付いていなかったのだ。<br>マズい『はぐれ』だ……！<br>そう思った時はもう遅かった。アスクの周囲に黒い瘴気が渦巻き、周りの樹木がバキバキと枯れて折れていく。その奥から現れたのは、赤い瞳をらんらんと輝かせ、口からどす黒い気を吐くドラゴンだった。傷だらけで吠え猛るその様子は、すっかり理性を失っている。<br>マスターと死に別れたり、何らかの理由で理性を失ってしまう竜をはぐれと呼ぶ。アスクは小さな頃から、これに出会ったら終わりだと、両親から繰り返し聞かされていた。<br>が、実際目の当たりにすると、その禍々しさはすさまじいもので、恐怖で足が凍ったように動かない。<br>「あ、これはダメだ。ここで死ぬのかな……オレ」そう思った瞬間「オイ」と言うのんきな声に正気に戻った。<br>アスクのズボンをつんつんと引っ張るオレンジの何かは、にっこりして再度赤い石を渡そうとしてくる。<br>「コレ、つかう。ナカマヨベル」<br>その石はただの石ではないのはすぐわかった。これは『ジェム』だ。そう、ドラゴンと人間が一生共にすると誓う契約の石……だがこんな大きな『ジェム』が存在するのか?<br>「ガアァアアアアアア」<br>迷う時間は無かった。<br>アスクが石を受け取り、天に掲げたその時、金の光に包まれたドラゴンが現れたのだった。
