# BLACK ROSEちほー7（blackrose7 区域）
::: info 区域简介
为了追踪现出天使本性并带走Ellie的Mastima，Aul一行人前往留存着天使传说的Teos岛。与此同时，养父Byron得知Louis正在调查天使和Makurīchi家的事情，于是追着他拜访了Makurīchi家。被各自的命运引导而聚集的他们，将会知晓怎样的真相？
:::
<details class='raw-text'>
<summary>查看日语原文 (Original Text)</summary>

以Makurīchi家为首，据说炼金术师始祖的家系诞生的Teos岛。那个岛上流传着传说：天使降临到受黑蔷薇病困扰的人们身边，赐予了各种奇迹的技艺。为了夺回Ellie并知晓Mastima的真正意图，Aul他们闯入了Teos岛。但是，从荒废的教会遗迹的痕迹中，Aul所分解的事实太过残酷。人们的命运是从何时起，又被谁所操纵的呢？Teos岛事件与黑翼的真相，不过是更大动荡的开始。
