# Phase 3（紫门）

::: warning 引文
何を失ったって　奇跡を求めていた

哪怕一无所有，依然相信会有奇迹

美しい日々と不完全　只、儚く

美丽的日子和不完全稍纵即逝

「変わらない」

「亘古不变」

その為に　その為に今　描いた

为了它 为了它，将它画出

「さあ、目覚めて」

来，醒来吧
:::

::: info 解锁条件

- 看到门的条件（解锁紫门的前提）：完走 [blackrose10 区域](/areas/blackrose10)（和白之门一样，完走之后可以发现紫之门）

- 拿到钥匙的条件：将 アウル 设为旅行伙伴的队长，在 1pc 内 **只** 游玩以下曲绘中含有 アウル 的歌曲，即不能游玩列表外的歌曲，否则该 pc 无效

::: tip 紫门有关区域
黑蔷薇区域：[blackrose3](/areas/blackrose3), [blackrose4](/areas/blackrose4), [blackrose5](/areas/blackrose5), [blackrose6](/areas/blackrose6), [blackrose7](/areas/blackrose7), [blackrose8](/areas/blackrose8), [blackrose9](/areas/blackrose9), [blackrose10](/areas/blackrose10) 区域
:::
